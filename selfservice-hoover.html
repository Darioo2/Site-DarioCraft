Qu'est-ce que SelfService Hoover ?

SelfService Hoover est un logiciel Windows qui récupère automatiquement votre planning de travail RATP depuis le portail SelfService (GIRO) et le synchronise avec votre Google Calendar.

Une fois configuré, le logiciel tourne en arrière-plan et met à jour votre agenda automatiquement deux fois par jour (5h et 14h), sans que vous ayez à faire quoi que ce soit.

Prérequis : un compte RATP actif (matricule DB + mot de passe), un compte Google, Windows 10 ou 11, une connexion internet.

Installation

  1. Téléchargez SelfService Hoover.exe ci-dessous.
  2. Lancez le fichier — aucune installation n'est requise. L'exécutable est autonome.
  3. Le logiciel s'installe automatiquement dans la barre des tâches (system tray, icône en bas à droite de l'écran).
  4. La fenêtre principale s'ouvre automatiquement à la première utilisation. Elle est accessible à tout moment en cliquant sur l'icône dans le tray.
Windows peut afficher un avertissement SmartScreen à la première exécution. Cliquez sur "Informations complémentaires" puis "Exécuter quand même".

Connexion RATP — Étape 1

Connectez votre compte portail SelfService RATP pour que le logiciel puisse récupérer votre planning.

  1. Ouvrez l'onglet Connexions → section Portail RATP.
  2. Saisissez votre matricule (ex : DB224810) et votre mot de passe.
  3. Cliquez Sauvegarder.
  4. Cliquez Tester la connexion pour vérifier que les identifiants sont corrects.
Les identifiants sont stockés de façon sécurisée dans Windows Credential Manager — le même système que vos mots de passe Wi-Fi et Windows. Ils ne sont jamais écrits en clair sur le disque.

Connexion Google Calendar — Étape 2

Autorisez le logiciel à accéder à votre Google Calendar via OAuth2 (aucun mot de passe Google n'est stocké).

  1. Onglet Connexions → section Google Calendar.
  2. Cliquez Connecter Google.
  3. Une page web s'ouvre dans votre navigateur. Connectez-vous avec votre compte Google et autorisez l'accès.
  4. Une fois autorisé, la page se ferme automatiquement et votre email apparaît dans l'application.
Le token OAuth2 est stocké dans Windows Credential Manager. Il peut être révoqué à tout moment depuis votre compte Google → Sécurité → Applications tierces.

Choisir l'agenda cible — Étape 3

Définissez dans quel agenda Google les événements de travail seront créés.

Recommandé

Créer un nouvel agenda "Planning RATP" — crée un agenda dédié, séparé de votre agenda personnel. Facile à activer/désactiver dans Google Calendar.

Avancé

Choisir un agenda existant — sélectionnez un agenda existant dans la liste déroulante. Les événements s'y ajouteront directement.

  1. Onglet Connexions → section Agenda cible.
  2. Choisissez l'option souhaitée.
  3. Cliquez Sauvegarder le choix d'agenda.

Lancer une synchronisation

Onglet Synchronisation

Sync complète (première fois ou remise à zéro)

  1. Onglet Synchronisation → section Synchronisation manuelle.
  2. Choisissez la plage : Personnalisé (ex : 300 jours) ou Planning complet.
  3. Cliquez Lancer la synchronisation.
  4. Suivez la progression dans l'onglet Journal.
Durée estimée : 5 à 10 minutes pour 10 mois de planning. La barre de progression affiche le jour en cours de traitement. L'onglet Journal détaille chaque événement créé ou mis à jour.

Sync rapide (usage quotidien)

Même procédure avec une plage de 7 jours — prend environ 1 à 2 minutes. Idéal pour vérifier que les prochains jours sont bien à jour dans Google Calendar.

Sync automatique

Configurez deux horaires (par défaut 5h00 et 14h00) dans la section Synchronisation automatique. Chaque horaire peut être activé/désactivé indépendamment.

Les dates de la dernière sync et de la prochaine sync auto sont affichées en bas de la section pour vérifier que tout tourne correctement.

Onglet Aperçu

Onglet Aperçu — Juin 2026

L'onglet Aperçu affiche votre planning sous forme de calendrier mensuel. Naviguez avec les flèches ◀ ▶ pour changer de mois.

Cliquez sur n'importe quel jour pour afficher le détail : code service, type, catégorie, horaires de début/fin.

Code couleur

Rose/Rouge — Service de jour
Violet — Service de nuit
Vert — Service mixte (jour + nuit)
Jaune — Formation ou congé (GCA)
Bleu — Repos
Les congés (GCA) sont uniquement visibles dans l'aperçu du logiciel (en jaune). Seuls les jours de travail sont exportés vers Google Calendar.

Icône dans le System Tray

L'icône en bas à droite de l'écran indique l'état des connexions :

Point ROUGE
Non connecté — RATP ou Google non configuré.
Point VERT
Connecté — les deux comptes sont actifs.

Menu clic droit

  • Ouvrir l'application — affiche la fenêtre principale
  • Lancer une synchronisation — lance une sync manuelle immédiate
  • Activer/désactiver la sync auto — bascule sans ouvrir l'app
  • Activer/désactiver la sourdine — coupe les notifications temporairement
  • Quitter — ferme complètement le logiciel

Paramètres

Thème
Sombre (recommandé) ou Clair. Accessible aussi via le bouton en haut à droite de la fenêtre.
Démarrage Windows
Lance le logiciel automatiquement au démarrage de Windows. Fortement recommandé pour profiter de la sync auto.
Démarrer réduit
Au démarrage, le logiciel se place directement dans le tray sans afficher la fenêtre principale.
Notification fin de sync
Affiche une notification Windows à la fin de chaque synchronisation avec le résumé des événements créés/modifiés.
Alerte si pas de MAJ
Envoie une alerte si aucune synchronisation n'a eu lieu depuis le nombre d'heures configuré (par défaut 48h).
Vitesse du scraper
Délai min/max entre les actions du navigateur invisible. Des délais plus longs rendent le scraping indiscernable d'un humain (recommandé : 2s / 4s).

Questions fréquentes

Le logiciel ouvre-t-il un navigateur pendant la sync ?

Non. Le navigateur est invisible (mode furtif). Vous verrez uniquement la progression dans l'onglet Journal.

Mes identifiants RATP sont-ils stockés de façon sécurisée ?

Oui. Ils sont stockés dans Windows Credential Manager, le même système que vos mots de passe Wi-Fi et Windows.

La sync modifie-t-elle mon agenda personnel Google ?

Non, si vous avez choisi "Créer un nouvel agenda Planning RATP". Seul cet agenda dédié est modifié.

Que se passe-t-il si mon mot de passe RATP change ?

Mettez à jour votre mot de passe dans l'onglet Connexions et relancez une synchronisation.

Les congés (GCA) apparaissent-ils dans Google Calendar ?

Non. Les congés sont uniquement visibles dans l'aperçu du logiciel (en jaune). Seuls les jours de travail sont exportés vers Google Calendar.

L'application consomme-t-elle beaucoup de ressources ?

Non. En dehors des syncs, elle consomme moins de 50 Mo de RAM et quasiment pas de CPU.

Données & confidentialité

Toutes les données sont stockées localement sur votre machine. Aucune donnée n'est envoyée vers un serveur tiers.

Fichiers locaux

%APPDATA%\SelfServiceHoover\session_cookies.json
Cookies de session RATP
Credential Manager · SelfServiceHoover_RATP
Identifiants RATP chiffrés
Credential Manager · SelfServiceHoover_GOOGLE
Token Google OAuth2

Communications réseau

Le logiciel communique uniquement avec :

  • Le portail RATP — selfservicerer.ratp.net
  • L'API Google Calendar — googleapis.com

Désinstallation complète

  1. Quittez l'application (clic droit tray → Quitter).
  2. Supprimez le fichier SelfService Hoover.exe.
  3. Supprimez le dossier %APPDATA%\SelfServiceHoover\ (optionnel).
  4. Dans le Gestionnaire d'informations d'identification Windows, supprimez les entrées SelfServiceHoover_RATP et SelfServiceHoover_GOOGLE (optionnel).
